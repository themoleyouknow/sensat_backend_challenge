Sensat Backend Challenge

An HTTP interface for reading sensor data from a database using Python and Flask.

How To Use:

Querying the Database

In order to query the database for box <box_id> for sensor readings between <from_date> and <to_date> then enter a url with the form:

http://localhost:5000/readings/<box_id>/<from_date>/<to_date>

Where:

  • <box_id> is the box id of the sensor box of interest
  • <from_date> is the datetime from which to search for readings of form: YYYY-MM-DD HH:MM:SS
  • <to_date> is the datetime to which to search for readings (inclusive) of form: YYYY-MM-DD HH:MM:SS

e.g. http://localhost:5000/readings/Box-A1/2019-01-01 00:00:00/2019-12-24 12:12:12

Changing Database Configuration Details

In order to edit the database configuration details, head to the configuration page to upload a .yaml file containing the relevant database configuration information to be changed.
